* Overview

Source repository for [[https://emacs.deno.dev][emacs.deno.dev]], powered by Deno Deploy.

* Getting Started

** Prerequisites

- [[https://deno.land][Deno]] runtime

** Development

Clone the repository and run locally:

#+begin_src shell
git clone https://github.com/conao3/deno-emacs.git
cd deno-emacs
#+end_src

* Project Structure

#+begin_example
deno-emacs/
  public/
    mod.js      # Main application entry point
  Makefile      # Build automation
  LICENSE       # MIT License
#+end_example

* Deployment

This project is deployed on [[https://deno.com/deploy][Deno Deploy]]. Push changes to the main branch to trigger automatic deployment.
